## Elm Chess
Study Project

## Install Instructions
- Install npm
- Install Elm (npm install -g elm)
- Instal Stylus (npm install -g stylus)

## Run Instructions
- run `make build`

## Debug Instructions
- run `elm-reactor`
- debugger will be avaliable at http://localhost:8000/index.html
